## 1. 存储过程（PROCEDURE）

### 🏠 生活类比

> **存储过程** = 写在厨房菜单上的做菜步骤：你点菜，厨房按步骤自动做（提前写好，调用执行）

### 📖 概念讲解

| 项目 | 说明 |
|------|------|
| 创建 | `CREATE PROCEDURE proc_name(params) BEGIN ... END` |
| 调用 | `CALL proc_name(args)` |
| 参数模式 | IN（输入）/ OUT（输出）/ INOUT（输入输出） |
| 特性 | 预编译、执行效率高、可包含复杂逻辑 |

### 💻 SQL 代码示例

```sql
CREATE DATABASE sp_demo;
USE sp_demo;

CREATE TABLE t_account (
    account_id INT PRIMARY KEY,
    account_name VARCHAR(50),
    balance DECIMAL(10,2)
);
INSERT INTO t_account VALUES (1, 'Alice', 10000), (2, 'Bob', 5000);

-- 创建存储过程：转账
DELIMITER //
CREATE PROCEDURE p_transfer(
    IN p_from_id INT,          -- 输入：转出账户
    IN p_to_id INT,            -- 输入：转入账户
    IN p_amount DECIMAL(10,2) -- 输入：转账金额
)
BEGIN
    -- 检查余额
    IF (SELECT balance FROM t_account WHERE account_id = p_from_id) < p_amount THEN
        SELECT 'Insufficient balance' AS message;
    ELSE
        -- 扣款
        UPDATE t_account SET balance = balance - p_amount WHERE account_id = p_from_id;
        -- 加款
        UPDATE t_account SET balance = balance + p_amount WHERE account_id = p_to_id;
        SELECT 'Transfer successful' AS message;
    END IF;
END//
DELIMITER ;

-- 调用存储过程
CALL p_transfer(1, 2, 1000);
SELECT * FROM t_account;
```

### ❌ 常见错误

- ❌ 忘记 DELIMITER // → 存储过程可能被分号截断
- ❌ 存储过程中写错参数名 → 调用时参数错位

## 2. 函数（FUNCTION）与存储过程的区别

### 📖 概念讲解

| 特性 | 存储过程 | 函数 |
|------|---------|------|
| 返回值 | 可返回多个结果集 | 必须有返回值（RETURNS） |
| 调用方式 | CALL | 在 SELECT/表达式中调用 |
| 参数模式 | IN/OUT/INOUT | 只能 IN |
| 修改数据 | ✅ 可以 | ❌ 不能 |
| 用途 | 复杂业务流程 | 计算返回值 |

### 💻 SQL 代码示例

```sql
USE sp_demo;

-- 创建函数：计算个人所得税
DELIMITER //
CREATE FUNCTION f_calculate_tax(p_salary DECIMAL(10,2))
RETURNS DECIMAL(10,2)
DETERMINISTIC
BEGIN
    DECLARE v_tax DECIMAL(10,2);
    IF p_salary <= 5000 THEN
        SET v_tax = 0;
    ELSEIF p_salary <= 10000 THEN
        SET v_tax = (p_salary - 5000) * 0.10;
    ELSE
        SET v_tax = (p_salary - 5000) * 0.20;
    END IF;
    RETURN v_tax;
END//
DELIMITER ;

-- 在查询中调用函数
SELECT emp_name, salary, f_calculate_tax(salary) AS tax FROM t_account;

-- 创建函数：返回格式化金额
DELIMITER //
CREATE FUNCTION f_format_amount(p_amount DECIMAL(10,2))
RETURNS VARCHAR(20)
DETERMINISTIC
BEGIN
    RETURN CONCAT('¥', FORMAT(p_amount, 2));
END//
DELIMITER ;

SELECT emp_name, f_format_amount(salary) AS salary_formatted
FROM t_account
WHERE f_calculate_tax(salary) > 100;
```

### ❌ 常见错误

- ❌ 在函数中尝试修改数据 → 报错

## 3. 变量、条件、循环

### 📖 概念讲解

| 项目 | 关键字 | 说明 |
|------|--------|------|
| 变量声明 | DECLARE | 在 BEGIN...END 中声明 |
| 变量赋值 | SET / SELECT ... INTO | 赋值 |
| IF 条件 | IF-THEN-ELSEIF-ELSE-END IF | 分支 |
| CASE | CASE-WHEN-THEN-ELSE-END CASE | 多分支 |
| WHILE 循环 | WHILE ... DO ... END WHILE | 条件循环 |
| REPEAT | REPEAT ... UNTIL ... END REPEAT | 至少执行一次 |
| LEAVE | LEAVE label | 跳出循环（类似 break） |
| ITERATE | ITERATE label | 继续下一次循环（类似 continue） |

### 💻 SQL 代码示例

```sql
USE sp_demo;

-- 变量演示
DELIMITER //
CREATE PROCEDURE p_variable_demo()
BEGIN
    DECLARE v_count INT DEFAULT 0;
    DECLARE v_max_salary DECIMAL(10,2);

    SELECT COUNT(*), MAX(salary) INTO v_count, v_max_salary FROM t_account;
    SELECT CONCAT('Total: ', v_count, ', Max: ', v_max_salary) AS result;
END//
DELIMITER ;

CALL p_variable_demo();

-- IF-THEN-ELSE 演示
DELIMITER //
CREATE FUNCTION f_grade_level(p_score INT)
RETURNS VARCHAR(10)
DETERMINISTIC
BEGIN
    DECLARE v_level VARCHAR(10);
    IF p_score >= 90 THEN
        SET v_level = 'A';
    ELSEIF p_score >= 80 THEN
        SET v_level = 'B';
    ELSEIF p_score >= 60 THEN
        SET v_level = 'C';
    ELSE
        SET v_level = 'D';
    END IF;
    RETURN v_level;
END//
DELIMITER ;

-- CASE 演示
DELIMITER //
CREATE FUNCTION f_month_name(p_month INT)
RETURNS VARCHAR(10)
DETERMINISTIC
BEGIN
    CASE p_month
        WHEN 1 THEN RETURN 'January';
        WHEN 2 THEN RETURN 'February';
        WHEN 3 THEN RETURN 'March';
        WHEN 4 THEN RETURN 'April';
        WHEN 5 THEN RETURN 'May';
        WHEN 6 THEN RETURN 'June';
        WHEN 7 THEN RETURN 'July';
        WHEN 8 THEN RETURN 'August';
        WHEN 9 THEN RETURN 'September';
        WHEN 10 THEN RETURN 'October';
        WHEN 11 THEN RETURN 'November';
        WHEN 12 THEN RETURN 'December';
        ELSE RETURN 'Invalid';
    END CASE;
END//
DELIMITER ;

SELECT f_month_name(3);  -- March

-- WHILE 循环演示
DELIMITER //
CREATE PROCEDURE p_while_demo()
BEGIN
    DECLARE i INT DEFAULT 1;
    DECLARE result VARCHAR(200) DEFAULT '';

    WHILE i <= 5 DO
        SET result = CONCAT(result, i, ',');
        SET i = i + 1;
    END WHILE;

    SELECT result AS sequence;
END//
DELIMITER ;

CALL p_while_demo();

-- REPEAT 循环（至少执行一次）
DELIMITER //
CREATE PROCEDURE p_repeat_demo()
BEGIN
    DECLARE i INT DEFAULT 1;
    DECLARE result VARCHAR(200) DEFAULT '';

    REPEAT
        SET result = CONCAT(result, i, ',');
        SET i = i + 1;
    UNTIL i > 5 END REPEAT;

    SELECT result AS sequence;
END//
DELIMITER ;

CALL p_repeat_demo();
```

### ❌ 常见错误

- ❌ 变量名拼写错误 → 变量未定义就使用

## 4. 游标（CURSOR）

### 🏠 生活类比

> **游标** = 手指指着列表的每一行：一行一行地看，处理完再看下一行

### 📖 概念讲解

游标用于在存储过程中逐行遍历查询结果集。

**使用步骤**：`DECLARE` → `OPEN` → `FETCH` → `CLOSE`

**注意**：
- 游标必须在变量声明之后声明
- 需要配合 `DECLARE ... HANDLER` 处理 NOT FOUND 条件

### 💻 SQL 代码示例

```sql
USE sp_demo;

CREATE TABLE t_sales (
    id INT PRIMARY KEY,
    product_name VARCHAR(50),
    amount DECIMAL(10,2)
);
INSERT INTO t_sales VALUES
    (1, 'Laptop', 5000), (2, 'Mouse', 99),
    (3, 'Keyboard', 299), (4, 'Monitor', 1299), (5, 'Headset', 199);

-- 使用游标统计销售总额
DELIMITER //
CREATE PROCEDURE p_cursor_demo()
BEGIN
    DECLARE done INT DEFAULT 0;
    DECLARE v_amount DECIMAL(10,2);
    DECLARE v_total DECIMAL(10,2) DEFAULT 0;

    -- 声明游标
    DECLARE cur CURSOR FOR SELECT amount FROM t_sales;

    -- 声明继续条件处理器（NOT FOUND 时设置 done=1）
    DECLARE CONTINUE HANDLER FOR NOT FOUND SET done = 1;

    OPEN cur;

    read_loop: LOOP
        FETCH cur INTO v_amount;
        IF done = 1 THEN
            LEAVE read_loop;
        END IF;
        SET v_total = v_total + v_amount;
    END LOOP;

    CLOSE cur;

    SELECT CONCAT('Total sales: ', v_total) AS result;
END//
DELIMITER ;

CALL p_cursor_demo();
```

### 📊 ASCII 游标流程

```
┌──────────────────────────────────┐
│ DECLARE cursor CURSOR FOR ...   │
│ DECLARE HANDLER FOR NOT FOUND   │
└──────────────────────────────────┘
           ↓
    ┌──────────────┐
    │  OPEN cursor  │   打开游标
    └──────────────┘
           ↓
    ┌──────────────┐
    │ FETCH INTO    │──► 处理数据
    │  done=1?     │
    └──────┬───────┘
           ↓
      ┌────────┐
      │ CLOSE  │   关闭游标
      └────────┘
```

### ❌ 常见错误

- ❌ 游标中忘记处理 NOT FOUND → 死循环

## 5. 查看、修改和删除

### 💻 SQL 代码示例

```sql
USE sp_demo;

-- 查看存储过程/函数列表
SHOW PROCEDURE STATUS;
SHOW FUNCTION STATUS;

-- 查看特定存储过程定义
SHOW CREATE PROCEDURE p_transfer;
SHOW CREATE FUNCTION f_calculate_tax;

-- 删除存储过程
DROP PROCEDURE IF EXISTS p_variable_demo;

-- 删除函数
DROP FUNCTION IF EXISTS f_format_amount;
```

### ❌ 常见错误

- ❌ 删除了还在被调用的存储过程 → 调用时会报错

## 本章小结

```
┌─────────────────────────────────────────────────────────┐
│              存储过程与函数核心                           │
├─────────────────────────────────────────────────────────┤
│ 存储过程 = 可执行复杂逻辑的代码块，CALL调用            │
│ 函数 = 必须有返回值，可在SELECT中使用                  │
│                                                          │
│ 存储过程：                                             │
│   CREATE PROCEDURE p(IN/OUT/INOUT params)             │
│   CALL p(args)                                        │
│                                                          │
│ 函数：                                                 │
│   CREATE FUNCTION f(params) RETURNS type             │
│   SELECT f(args) 或 WHERE f(args) > 10              │
│                                                          │
│ 控制流：IF/CASE/WHILE/REPEAT/LEAVE/ITERATE           │
│ 游标：DECLARE → OPEN → FETCH → CLOSE               │
│                                                          │
│ 特性声明：DETERMINISTIC / NO SQL / READS SQL DATA   │
└─────────────────────────────────────────────────────────┘
```
