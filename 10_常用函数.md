## 1. 字符串函数

### 🏠 生活类比

> **字符串函数** = 文字处理工具箱：可以拼接、截取、大小写转换、去除空格

### 📖 概念讲解

| 函数 | 说明 |
|------|------|
| CONCAT(s1, s2, ...) | 连接字符串 |
| CONCAT_WS(sep, s1, s2) | 指定分隔符连接 |
| LENGTH(str) | 字节长度 |
| CHAR_LENGTH(str) | 字符数 |
| LEFT(str, n) / RIGHT(str, n) | 左侧/右侧截取 |
| SUBSTRING(str, pos, len) | 截取字符串（pos从1开始） |
| UPPER(str) / LOWER(str) | 转大写/小写 |
| TRIM(str) / LTRIM / RTRIM | 去除首尾空格 |
| REPLACE(str, old, new) | 替换字符串 |

### 💻 SQL 代码示例

```sql
CREATE DATABASE functions_demo;
USE functions_demo;

-- CONCAT：连接字符串
SELECT CONCAT('Hello', ' ', 'World') AS greeting;
SELECT CONCAT('User_', 123) AS username;

-- CONCAT_WS：指定分隔符连接
SELECT CONCAT_WS('-', '2024', '01', '15') AS date_str;  -- 2024-01-15

-- LENGTH vs CHAR_LENGTH
SELECT
    LENGTH('中国') AS byte_len,        -- 6（utf8mb4 每个汉字4字节）
    CHAR_LENGTH('中国') AS char_len;   -- 2（2个字符）

-- 截取函数
SELECT LEFT('HelloWorld', 5) AS left_part;    -- Hello
SELECT RIGHT('HelloWorld', 5) AS right_part; -- World
SELECT SUBSTRING('HelloWorld', 6, 5) AS mid;  -- World（从第6个字符开始，取5个）

-- 大小写转换
SELECT UPPER('hello') AS upper_str;  -- HELLO
SELECT LOWER('HELLO') AS lower_str;  -- hello

-- TRIM：去除首尾空格
SELECT CONCAT('[', TRIM('  hello  '), ']') AS trimmed;  -- [hello]

-- REPLACE：替换字符串
SELECT REPLACE('Hello World', 'World', 'MySQL') AS replaced;

-- 实战：脱敏手机号
CREATE TABLE t_user_info (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50),
    phone VARCHAR(20),
    email VARCHAR(100)
);
INSERT INTO t_user_info VALUES
    (1, 'alice', '13812345678', 'ALICE@EXAMPLE.COM'),
    (2, 'bob', '13998765432', 'BOB@EXAMPLE.COM');

-- 拼接
SELECT CONCAT(username, ' - ', phone) AS contact FROM t_user_info;
-- 脱敏手机号：138****5678
SELECT username,
       CONCAT(SUBSTRING(phone, 1, 3), '****', SUBSTRING(phone, 8)) AS masked_phone
FROM t_user_info;
-- 邮箱转小写
SELECT username, LOWER(email) AS email FROM t_user_info;
```

### ❌ 常见错误

- ❌ 混淆 LENGTH 和 CHAR_LENGTH → 中文场景下汉字字节数和字符数不同
- ❌ SUBSTRING(str, 0, len) → MySQL 字符串从 1 开始，不是 0

## 2. 数值函数

### 📖 概念讲解

| 函数 | 说明 |
|------|------|
| ABS(n) | 绝对值 |
| CEIL(n) / CEILING(n) | 向上取整 |
| FLOOR(n) | 向下取整 |
| ROUND(n, d) | 四舍五入 |
| TRUNCATE(n, d) | 截断（不四舍五入） |
| POW(n, e) / POWER(n, e) | 指数 |
| SQRT(n) | 平方根 |
| n DIV m | 整数除法 |
| n % m / MOD(n, m) | 取余 |

### 💻 SQL 代码示例

```sql
USE functions_demo;

SELECT 10 + 3 AS add_result;
SELECT 10 - 3 AS sub_result;
SELECT 10 * 3 AS mul_result;
SELECT 10 / 3 AS div_result;           -- 精确小数
SELECT 10 DIV 3 AS int_div_result;    -- 3（整数除法）
SELECT 10 % 3 AS mod_result;           -- 1（余数）

SELECT ABS(-10) AS abs_result;         -- 10
SELECT ROUND(3.14159, 2) AS rounded;   -- 3.14
SELECT CEIL(3.1) AS ceil_result;       -- 4
SELECT FLOOR(3.9) AS floor_result;     -- 3
SELECT TRUNCATE(3.14159, 3) AS truncated;  -- 3.141（截断）

SELECT SQRT(16) AS sqrt_result;        -- 4
SELECT POW(2, 3) AS pow_result;        -- 8

-- 实战：计算折扣价
CREATE TABLE t_product_price (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    price DECIMAL(10,2)
);
INSERT INTO t_product_price VALUES
    (1, 'Laptop', 5000.00), (2, 'Mouse', 89.50);

SELECT name, price, ROUND(price * 0.85, 2) AS discounted_price FROM t_product_price;
```

### ❌ 常见错误

- ❌ 以为 10/3 = 3 → MySQL 中 / 返回精确小数，DIV 才返回整数

## 3. 日期时间函数

### 📖 概念讲解

| 函数 | 说明 |
|------|------|
| NOW() / SYSDATE() | 当前日期时间 |
| CURDATE() | 当前日期 |
| CURTIME() | 当前时间 |
| YEAR() / MONTH() / DAY() | 提取年月日 |
| DATE_ADD(date, INTERVAL expr unit) | 日期加 |
| DATE_SUB(date, INTERVAL expr unit) | 日期减 |
| DATEDIFF(date1, date2) | 日期差（天） |
| TIMESTAMPDIFF(unit, date1, date2) | 时间差（精确到单位） |
| DATE_FORMAT(date, format) | 日期格式化 |

### 💻 SQL 代码示例

```sql
USE functions_demo;

SELECT NOW() AS current_datetime;   -- 2024-01-15 12:30:45
SELECT CURDATE() AS current_date;   -- 2024-01-15
SELECT CURTIME() AS current_time;   -- 12:30:45

SELECT
    YEAR(NOW()) AS yr,
    MONTH(NOW()) AS mo,
    DAY(NOW()) AS dy,
    HOUR(NOW()) AS hr,
    MINUTE(NOW()) AS mi,
    SECOND(NOW()) AS sc;

-- 日期加减
SELECT DATE_ADD('2024-01-15', INTERVAL 10 DAY) AS after_10_days;
SELECT DATE_SUB('2024-01-15', INTERVAL 1 MONTH) AS before_1_month;
SELECT DATE_ADD(NOW(), INTERVAL 1 HOUR) AS after_1_hour;

-- 日期差
SELECT DATEDIFF('2024-01-20', '2024-01-15') AS days_diff;       -- 5
SELECT TIMESTAMPDIFF(DAY, '2024-01-15', '2024-01-20') AS days_diff2;  -- 5
SELECT TIMESTAMPDIFF(HOUR, '2024-01-15 10:00', '2024-01-15 12:00') AS hours_diff;  -- 2

-- 实战：计算员工在职时长
CREATE TABLE t_employee_hired (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    hire_date DATE
);
INSERT INTO t_employee_hired VALUES
    (1, 'Alice', '2022-03-15'),
    (2, 'Bob', '2023-07-20'),
    (3, 'Carol', '2024-01-10');

SELECT
    name,
    hire_date,
    DATEDIFF(CURDATE(), hire_date) AS days_employed,
    TIMESTAMPDIFF(YEAR, hire_date, CURDATE()) AS years_employed
FROM t_employee_hired;
```

### ❌ 常见错误

- ❌ DATEDIFF vs TIMESTAMPDIFF → DATEDIFF 只算天，TIMESTAMPDIFF 可指定单位

## 4. DATE_FORMAT 和 STR_TO_DATE

### 📖 概念讲解

DATE_FORMAT 常用格式符：

| 格式符 | 说明 | 示例 |
|--------|------|------|
| %Y | 4位年 | 2024 |
| %y | 2位年 | 24 |
| %m | 2位月 | 01 |
| %d | 2位日 | 15 |
| %H | 24小时制 | 14 |
| %h | 12小时制 | 02 |
| %i | 分钟 | 30 |
| %s | 秒 | 45 |
| %W | 星期全名 | Monday |
| %a | 星期缩写 | Mon |

### 💻 SQL 代码示例

```sql
USE functions_demo;

SELECT DATE_FORMAT(NOW(), '%Y-%m-%d') AS date_iso;          -- 2024-01-15
SELECT DATE_FORMAT(NOW(), '%Y年%m月%d日') AS date_cn;     -- 2024年01月15日
SELECT DATE_FORMAT(NOW(), '%Y-%m-%d %H:%i:%s') AS datetime_full;
SELECT DATE_FORMAT(NOW(), '%H:%i:%s') AS time_24h;
SELECT DATE_FORMAT(NOW(), '%W') AS weekday_name;            -- Monday

-- 解析字符串为日期
SELECT STR_TO_DATE('2024-01-15', '%Y-%m-%d') AS parsed_date;
SELECT STR_TO_DATE('15/01/2024', '%d/%m/%Y') AS parsed_date_eu;

-- 实战：订单时间格式化
CREATE TABLE t_order_info (
    order_id INT PRIMARY KEY,
    order_date DATETIME
);
INSERT INTO t_order_info VALUES
    (1, '2024-01-15 14:30:00'), (2, '2024-02-20 09:15:00');

SELECT
    order_id,
    DATE_FORMAT(order_date, '%Y年%m月%d日 %H:%i') AS order_datetime_cn
FROM t_order_info;
```

## 5. 流程控制函数

### 📖 概念讲解

| 函数 | 说明 |
|------|------|
| IF(cond, val_true, val_false) | 条件表达式 |
| IFNULL(expr, default) | NULL 替换 |
| COALESCE(v1, v2, ...) | 返回第一个非 NULL 值 |
| CASE WHEN ... END | 多条件分支 |

### 💻 SQL 代码示例

```sql
USE functions_demo;

-- IF 条件表达式
SELECT IF(10 > 5, 'Yes', 'No') AS result;

CREATE TABLE t_student_grade (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    score INT
);
INSERT INTO t_student_grade VALUES
    (1, 'Alice', 85), (2, 'Bob', 72), (3, 'Carol', 93);

-- 嵌套 IF 转为等级
SELECT
    name, score,
    IF(score >= 90, 'A',
      IF(score >= 80, 'B',
        IF(score >= 60, 'C', 'D'))) AS grade
FROM t_student_grade;

-- IFNULL
SELECT IFNULL(NULL, 'default') AS result;      -- default
SELECT IFNULL('value', 'default') AS result;  -- value

-- COALESCE：返回第一个非 NULL 值
SELECT COALESCE(NULL, NULL, 'first', 'second') AS result;  -- first

-- 实战：获取联系方式（优先用手机，其次邮箱，再次微信）
CREATE TABLE t_contact (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    phone VARCHAR(20),
    email VARCHAR(100),
    wechat VARCHAR(50)
);
INSERT INTO t_contact VALUES
    (1, 'Alice', '13812345678', 'alice@example.com', 'alice_wx'),
    (2, 'Bob', NULL, 'bob@example.com', NULL),
    (3, 'Carol', '13900000000', NULL, NULL);

SELECT name, COALESCE(phone, email, wechat, 'No Contact') AS contact FROM t_contact;

-- CASE WHEN：多条件分支
CREATE TABLE t_sales_region (
    region VARCHAR(20),
    amount DECIMAL(10,2)
);
INSERT INTO t_sales_region VALUES
    ('North', 5000), ('South', 3000), ('East', 4000), ('West', 2000);

SELECT
    SUM(CASE WHEN region = 'North' THEN amount ELSE 0 END) AS north_total,
    SUM(CASE WHEN region = 'South' THEN amount ELSE 0 END) AS south_total,
    SUM(CASE WHEN region = 'East' THEN amount ELSE 0 END) AS east_total,
    SUM(CASE WHEN region = 'West' THEN amount ELSE 0 END) AS west_total
FROM t_sales_region;
```

### ❌ 常见错误

- ❌ IFNULL 和 COALESCE 混用 → IFNULL 只接受2个参数，COALESCE 可接受多个

## 6. GROUP_CONCAT（MySQL 特有）

### 🏠 生活类比

> **GROUP_CONCAT** = 把每个班级的学生名单用逗号连成一行

### 📖 概念讲解

`GROUP_CONCAT(col [ORDER BY col] [SEPARATOR 'delimiter'])`

- 默认分隔符：逗号
- 可配合 DISTINCT 去重、ORDER BY 排序
- 最大返回值长度由 group_concat_max_len 控制

### 💻 SQL 代码示例

```sql
USE functions_demo;

CREATE TABLE t_student_subject (
    id INT PRIMARY KEY AUTO_INCREMENT,
    student_name VARCHAR(50),
    subject VARCHAR(20)
);
INSERT INTO t_student_subject VALUES
    (1, 'Alice', 'Math'),
    (2, 'Alice', 'English'),
    (3, 'Alice', 'Physics'),
    (4, 'Bob', 'Math'),
    (5, 'Bob', 'English'),
    (6, 'Carol', 'Math');

-- 基本用法
SELECT student_name, GROUP_CONCAT(subject) AS subjects
FROM t_student_subject
GROUP BY student_name;

-- 指定分隔符和排序
SELECT student_name,
       GROUP_CONCAT(subject ORDER BY subject SEPARATOR ', ') AS subjects
FROM t_student_subject
GROUP BY student_name;

-- 实战：订单商品列表
CREATE TABLE t_order_product (
    order_id INT,
    product_name VARCHAR(50)
);
INSERT INTO t_order_product VALUES
    (1, 'Laptop'), (1, 'Mouse'), (1, 'Keyboard'),
    (2, 'Monitor'),
    (3, 'Keyboard'), (3, 'Webcam');

SELECT order_id, GROUP_CONCAT(product_name SEPARATOR ', ') AS product_list
FROM t_order_product
GROUP BY order_id;
```

## 7. CAST 和 CONVERT

### 📖 概念讲解

| 函数 | 说明 |
|------|------|
| CAST(expr AS TYPE) | 类型转换（标准 SQL） |
| CONVERT(expr, TYPE) | 类型转换（MySQL 特有，支持字符集） |

常见类型：CHAR / DATE / DATETIME / DECIMAL / SIGNED / UNSIGNED

### 💻 SQL 代码示例

```sql
USE functions_demo;

SELECT CAST('123' AS SIGNED) + 10 AS result;           -- 133
SELECT CONVERT('123.45', DECIMAL(5,2)) AS result;    -- 123.45
SELECT CAST(123 AS CHAR) AS str_val;                  -- '123'
SELECT CAST('2024-01-15' AS DATE) AS date_val;       -- 2024-01-15

-- 实战：字符串金额求和
CREATE TABLE t_sales_str (
    id INT PRIMARY KEY,
    amount VARCHAR(20)
);
INSERT INTO t_sales_str VALUES (1, '1000.50'), (2, '2000.75');

SELECT SUM(CAST(amount AS DECIMAL(10,2))) AS total FROM t_sales_str;
```
